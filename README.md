# Physiotherapy OPD Services Website

### GD Goenka University

---

## Overview

The Physiotherapy OPD Services Website is a full-stack web application designed to streamline outpatient department (OPD) services for a physiotherapy unit. This platform enables patients to book appointments, submit medical details, and make secure payments, while allowing administrators and physiotherapists to manage patient records and schedules efficiently.

---

## Objective

To develop a professional, user-friendly, and secure web platform that digitizes physiotherapy OPD services and improves patient experience, appointment management, and clinical workflow.

---

## Features

### Patient Features

* Online Appointment Booking
* Patient Intake Form (Medical & Personal Details)
* Secure Payment Integration
* Patient Dashboard
* Appointment History
* Upload Medical Reports
* Download Prescriptions / Exercise Plans

---

### Admin Features

* Manage Appointments (Approve/Cancel)
* Add/Edit Physiotherapists

### Physiotherapist Features

* View Assigned Appointments
